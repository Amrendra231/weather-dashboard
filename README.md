# 🌤 Weather Dashboard

A responsive Weather Dashboard built using HTML, CSS and Vanilla JavaScript.

## Features
- 🔍 Search weather by city
- 🌡 Current temperature
- 💧 Humidity
- 💨 Wind speed
- 🤗 Feels like temperature
- 📱 Responsive Design
- ⚡ Fast Weather API
- - 🌍 Search weather for any city
- 🚀 No API Key Required (Powered by Open-Meteo)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
